# 📄 Resume Analyzer

A simple Resume Analyzer web app built using Python and Streamlit.

## 🚀 Features
- Upload resume (.txt file)
- Manual resume text input
- Detects skills (Python, Java, SQL, ML, Data Analysis)
- Calculates resume score
- Provides suggestions for improvement

## 🛠️ Tech Stack
- Python
- Streamlit

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run resume.py

## 📌 Example
Input:
"I know python and machine learning"

Output:
- Skills found: python, machine learning
- Resume Score: 40%
- Suggestions: add java, sql, data analysis
